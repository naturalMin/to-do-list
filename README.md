# To Do List
deployment : https://naturalmin.github.io/to-do-list/
### 사용기술
* React.js
* TypeScript
* styled-component
* React-Helmet
* Recoil Library 
* React Hook Form
* LocalStorage

### 구현요소
* Task List 1page로 구성
* 4개의 Component로 구성: 
  * add to do 기능
  * select 기능
  * delete to do 기능
  * to do list UI
* atom selector 활용 : custom category구현
* localStorage 활용 : to do set, get, remove 기능 구현해서 데이터 손실 방지.
